# Git assignment

## How I Will Use Git and VS Code
I will use Git to track changes in my project by initializing a repository,
creating branches, and committing my work at each stage.
VS Code will be my code editor, where I will create and edit files,
and use the built-in terminal to run Git commands.

## What This Assignment Demonstrates About GitHub
This assignment demonstrates how GitHub is used as a remote hosting platform
for Git repositories. It shows how to push local code to GitHub, collaborate
using branches and pull requests, and manage code reviews — all core skills
in real-world software development.
